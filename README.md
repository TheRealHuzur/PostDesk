# PostDesk

PostDesk ist ein praktisches Tool zur einfachen Formatierung deiner Texte für LinkedIn. 

Es konvertiert klassischen Markdown-Text (wie **fett**, *kursiv*, oder Blockzitate `> `) mithilfe von Unicode-Zeichen in eine Form, die du direkt kopieren und in deine LinkedIn-Beiträge einfügen kannst.

## Funktionen
- **Automatisches Formatieren**: Verwandelt Markdown-Formatierungen in entsprechende Unicode-Alternativen, die auf LinkedIn dargestellt werden können.
- **Fehlerfreie Darstellung**: Nutzt speziell ausgesuchte Unicode-Blöcke (Mathematical Sans-Serif), bei denen keine Buchstaben (wie 'h' oder 'e') fehlen.
- **Kopieren mit einem Klick**: Ein eingebauter "Kopieren"-Button überträgt das fertige Ergebnis sicher in deine Zwischenablage.
- **Dark- / Light-Mode**: Wähle dein bevorzugtes Farbschema über den Toggle im Header.

## Bedienung
1. Öffne die `index.html` in deinem Webbrowser.
2. Füge deinen formatierten Text im Bereich **Eingabe** (Roher Text) ein.
3. Überprüfe das fertige LinkedIn-Format im Bereich **Output**.
4. Klicke auf **Kopieren**, um den Text für deinen nächsten Post in die Zwischenablage zu legen.
